
# Healthcare Analytics Portfolio (Anonymized)

This repo publishes a GitHub Pages site from `/docs` with a 10‑year patient‑year dataset.
Use **Actions → Render charts & stats** to refresh figures and tables after you swap in new data.

## Quick start
1) Settings → Pages → Source: `Deploy from a branch`, Branch: `main` / `docs`.
2) Visit the Pages URL shown after the first build.
